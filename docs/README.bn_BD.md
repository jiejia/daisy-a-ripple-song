# A Ripple Song

A Ripple Song একটি ক্লাসিক WordPress থিম। এর সঙ্গে থাকা [পডকাস্ট প্লাগইন](https://github.com/jiejia/a-ripple-song-podcast) ইনস্টল করলে অডিও প্লেয়ার, পডকাস্ট উইজেট এবং সংশ্লিষ্ট ফিচারগুলো চালু করা যায়।

<img width="" src="../screenshot.png"/>

## বৈশিষ্ট্য

- 56টি সুন্দর থিম রঙের স্কিম (26টি লাইট থিম এবং 26টি ডার্ক থিম)
- আন্তর্জাতিকীকরণ সমর্থন
- ডেটা ট্র্যাকিং
- Swup ভিত্তিক মসৃণ পেজ ট্রানজিশন
- থিমের শৈলীর সাথে মানানসই একাধিক উইজেট (কিছু ফিচারের জন্য পডকাস্ট প্লাগইন দরকার)
- নিমগ্ন অডিও প্লেয়ার (পডকাস্ট প্লাগইন দরকার)
