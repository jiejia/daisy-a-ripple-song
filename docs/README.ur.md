# A Ripple Song

A Ripple Song ایک کلاسک WordPress تھیم ہے جس میں آڈیو پلیئر، پوڈکاسٹ صفحات اور ویجٹس پہلے سے شامل ہیں۔ [پوڈکاسٹ پلگ ان](https://github.com/jiejia/a-ripple-song-podcast) کے ساتھ مل کر یہ پوڈکاسٹ کی اشاعت اور مینجمنٹ، اور بڑے پوڈکاسٹ پلیٹ فارمز کے ساتھ ہم آہنگی کے لیے Podcast RSS بنانے کی سہولت دیتا ہے۔

<img width="" src="../screenshot.png"/>

## خصوصیات

- 56 خوبصورت تھیم رنگ اسکیمیں (26 لائٹ اور 26 ڈارک تھیمز)
- غرق کر دینے والا آڈیو پلیئر
- بین الاقوامی زبانوں کی سپورٹ
- ڈیٹا ٹریکنگ
- Swup پر مبنی ہموار صفحہ منتقلی
- تھیم کے انداز سے مطابقت رکھنے والے متعدد ویجٹس
