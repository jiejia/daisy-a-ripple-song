# A Ripple Song

A Ripple Song هو قالب WordPress كلاسيكي. بعد تثبيت [إضافة البودكاست المرافقة](https://github.com/jiejia/a-ripple-song-podcast) يمكنك تفعيل مشغل الصوت وودجات البودكاست والميزات المرتبطة بها.

<img width="" src="../screenshot.png"/>

## المميزات

- 56 لوحة ألوان أنيقة للقالب (26 سمة فاتحة و26 سمة داكنة)
- دعم التدويل
- تتبع البيانات
- انتقالات صفحات سلسة عبر Swup
- عدة ودجات متناسقة مع أسلوب القالب (بعض الوظائف تتطلب إضافة البودكاست)
- مشغل صوت غامر (يتطلب إضافة البودكاست)

## التقنيات المستخدمة

- [Vite](https://github.com/vitejs/vite) - أداة لتطوير وبناء الواجهة الأمامية
- [Tailwind CSS](https://github.com/tailwindlabs/tailwindcss) - إطار CSS قائم على الأدوات المساعدة
- [daisyUI](https://github.com/saadeghi/daisyui) - مكتبة مكونات UI مبنية على Tailwind CSS
- [Alpine.js](https://github.com/alpinejs/alpine) - إطار خفيف للتفاعلات في الواجهة الأمامية
- [Lucide](https://github.com/lucide-icons/lucide) - مكتبة أيقونات مفتوحة المصدر
- [Simple Icons](https://github.com/simple-icons/simple-icons) - مكتبة أيقونات العلامات التجارية
- [Swup](https://github.com/swup/swup) - مكتبة انتقالات الصفحات
- [Howler.js](https://github.com/goldfire/howler.js) - مكتبة تشغيل الصوت
- [audioMotion-analyzer](https://github.com/hvianna/audioMotion-analyzer) - مكتبة لعرض الطيف الصوتي بصرياً
- [Carbon Fields](https://github.com/htmlburger/carbon-fields) - إطار حقول مخصصة لـ WordPress
- [PHP-Scoper](https://github.com/humbug/php-scoper) - أداة لعزل نطاقات أسماء PHP
