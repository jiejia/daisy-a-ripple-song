# Daisy A Ripple Song

Daisy A Ripple Song은 클래식 WordPress 테마입니다. [팟캐스트 플러그인](https://github.com/jiejia/a-ripple-song)을 설치하면 오디오 플레이어와 팟캐스트 위젯 등 관련 기능을 사용할 수 있습니다.

<img width="" src="../screenshot.png"/>

## Demo

[https://podcast.aripplesong.me/](https://podcast.aripplesong.me/)

## 특징

- 56가지의 세련된 테마 색상 조합(26가지 라이트 테마와 26가지 다크 테마)
- 국제화 지원
- 데이터 추적
- Swup 기반의 매끄러운 페이지 전환
- 테마 스타일에 어울리는 다양한 widgets(일부 기능은 팟캐스트 플러그인 필요)
- 몰입감 있는 오디오 플레이어(팟캐스트 플러그인 필요)

## 기술 스택

- [Vite](https://github.com/vitejs/vite) - 프런트엔드 개발 및 빌드 도구
- [Tailwind CSS](https://github.com/tailwindlabs/tailwindcss) - 유틸리티 우선 CSS 프레임워크
- [daisyUI](https://github.com/saadeghi/daisyui) - Tailwind CSS 기반 UI 컴포넌트 라이브러리
- [Alpine.js](https://github.com/alpinejs/alpine) - 경량 프런트엔드 인터랙션 프레임워크
- [Lucide](https://github.com/lucide-icons/lucide) - 오픈소스 아이콘 라이브러리
- [Simple Icons](https://github.com/simple-icons/simple-icons) - 브랜드 아이콘 라이브러리
- [Swup](https://github.com/swup/swup) - 페이지 전환 라이브러리
- [Howler.js](https://github.com/goldfire/howler.js) - 오디오 재생 라이브러리
- [audioMotion-analyzer](https://github.com/hvianna/audioMotion-analyzer) - 오디오 스펙트럼 시각화 라이브러리
- [Carbon Fields](https://github.com/htmlburger/carbon-fields) - WordPress 커스텀 필드 프레임워크
- [PHP-Scoper](https://github.com/humbug/php-scoper) - PHP 네임스페이스 격리 도구
