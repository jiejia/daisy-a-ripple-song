# A Ripple Song

A Ripple Song 是一款經典 WordPress 佈景主題，安裝配套[播客外掛](https://github.com/jiejia/a-ripple-song-podcast)後，即可啟用音訊播放器、播客小工具等功能。

<img width="" src="../screenshot.png"/>

## 特性

- 56 套精美主題配色（26 套亮色主題和 26 套暗色主題）
- 國際化支援
- 數據追蹤
- 基於 Swup 的無刷新頁面切換技術
- 多個符合主題風格的小工具（部分功能需安裝播客外掛）
- 沉浸式音訊播放器（需安裝播客外掛）

## 技術棧

- [Vite](https://github.com/vitejs/vite) - 前端開發與建構工具
- [Tailwind CSS](https://github.com/tailwindlabs/tailwindcss) - 原子化 CSS 框架
- [daisyUI](https://github.com/saadeghi/daisyui) - 基於 Tailwind CSS 的 UI 元件庫
- [Alpine.js](https://github.com/alpinejs/alpine) - 輕量級前端互動框架
- [Lucide](https://github.com/lucide-icons/lucide) - 開源圖示庫
- [Simple Icons](https://github.com/simple-icons/simple-icons) - 品牌圖示庫
- [Swup](https://github.com/swup/swup) - 無刷新頁面切換函式庫
- [Howler.js](https://github.com/goldfire/howler.js) - 音訊播放函式庫
- [audioMotion-analyzer](https://github.com/hvianna/audioMotion-analyzer) - 音訊頻譜視覺化函式庫
- [Carbon Fields](https://github.com/htmlburger/carbon-fields) - WordPress 自訂欄位框架
- [PHP-Scoper](https://github.com/humbug/php-scoper) - PHP 命名空間隔離工具
