# A Ripple Song

A Ripple Song هو قالب WordPress كلاسيكي يأتي مع مشغل صوت وصفحات بودكاست وودجات مدمجة. وعند استخدامه مع [إضافة البودكاست](https://github.com/jiejia/a-ripple-song-podcast) يمكنك نشر البودكاست وإدارته، وإنشاء Podcast RSS لمزامنته مع أبرز منصات البودكاست.

<img width="" src="../screenshot.png"/>

## المميزات

- 56 لوحة ألوان أنيقة للقالب (26 سمة فاتحة و26 سمة داكنة)
- مشغل صوت غامر
- دعم التدويل
- تتبع البيانات
- انتقالات صفحات سلسة عبر Swup
- عدة ودجات متناسقة مع أسلوب القالب
