# Daisy A Ripple Song

Daisy A Ripple Song एक क्लासिक WordPress थीम है। [पॉडकास्ट प्लगइन](https://github.com/jiejia/a-ripple-song) इंस्टॉल करने के बाद आप ऑडियो प्लेयर, पॉडकास्ट विजेट्स और संबंधित फीचर्स इस्तेमाल कर सकते हैं।

<img width="" src="../screenshot.png"/>

## Demo

[https://podcast.aripplesong.me/](https://podcast.aripplesong.me/)

## विशेषताएं

- 56 खूबसूरती से तैयार की गई थीम रंग योजनाएं (26 लाइट और 26 डार्क थीम)
- अंतरराष्ट्रीयकरण समर्थन
- डेटा ट्रैकिंग
- Swup आधारित निर्बाध पेज ट्रांजिशन
- थीम की शैली के अनुरूप कई widgets (कुछ फीचर्स के लिए पॉडकास्ट प्लगइन जरूरी है)
- इमर्सिव ऑडियो प्लेयर (पॉडकास्ट प्लगइन जरूरी है)

## तकनीकी स्टैक

- [Vite](https://github.com/vitejs/vite) - फ्रंटएंड डेवलपमेंट और बिल्ड टूल
- [Tailwind CSS](https://github.com/tailwindlabs/tailwindcss) - यूटिलिटी-फर्स्ट CSS फ्रेमवर्क
- [daisyUI](https://github.com/saadeghi/daisyui) - Tailwind CSS पर आधारित UI कंपोनेंट लाइब्रेरी
- [Alpine.js](https://github.com/alpinejs/alpine) - हल्का फ्रंटएंड इंटरैक्शन फ्रेमवर्क
- [Lucide](https://github.com/lucide-icons/lucide) - ओपन सोर्स आइकन लाइब्रेरी
- [Simple Icons](https://github.com/simple-icons/simple-icons) - ब्रांड आइकन लाइब्रेरी
- [Swup](https://github.com/swup/swup) - पेज ट्रांजिशन लाइब्रेरी
- [Howler.js](https://github.com/goldfire/howler.js) - ऑडियो प्लेबैक लाइब्रेरी
- [audioMotion-analyzer](https://github.com/hvianna/audioMotion-analyzer) - ऑडियो स्पेक्ट्रम विजुअलाइजेशन लाइब्रेरी
- [Carbon Fields](https://github.com/htmlburger/carbon-fields) - WordPress कस्टम फील्ड फ्रेमवर्क
- [PHP-Scoper](https://github.com/humbug/php-scoper) - PHP नेमस्पेस आइसोलेशन टूल
