# A Ripple Song

A Ripple Song একটি ক্লাসিক WordPress থিম। এর সঙ্গে থাকা [পডকাস্ট প্লাগইন](https://github.com/jiejia/a-ripple-song-podcast) ইনস্টল করলে অডিও প্লেয়ার, পডকাস্ট উইজেট এবং সংশ্লিষ্ট ফিচারগুলো চালু করা যায়।

<img width="" src="../screenshot.png"/>

## বৈশিষ্ট্য

- 56টি সুন্দর থিম রঙের স্কিম (26টি লাইট থিম এবং 26টি ডার্ক থিম)
- আন্তর্জাতিকীকরণ সমর্থন
- ডেটা ট্র্যাকিং
- Swup ভিত্তিক মসৃণ পেজ ট্রানজিশন
- থিমের শৈলীর সাথে মানানসই একাধিক উইজেট (কিছু ফিচারের জন্য পডকাস্ট প্লাগইন দরকার)
- নিমগ্ন অডিও প্লেয়ার (পডকাস্ট প্লাগইন দরকার)

## প্রযুক্তি স্ট্যাক

- [Vite](https://github.com/vitejs/vite) - ফ্রন্টএন্ড ডেভেলপমেন্ট ও বিল্ড টুল
- [Tailwind CSS](https://github.com/tailwindlabs/tailwindcss) - ইউটিলিটি-ফার্স্ট CSS ফ্রেমওয়ার্ক
- [daisyUI](https://github.com/saadeghi/daisyui) - Tailwind CSS ভিত্তিক UI কম্পোনেন্ট লাইব্রেরি
- [Alpine.js](https://github.com/alpinejs/alpine) - হালকা ফ্রন্টএন্ড ইন্টারঅ্যাকশন ফ্রেমওয়ার্ক
- [Lucide](https://github.com/lucide-icons/lucide) - ওপেন সোর্স আইকন লাইব্রেরি
- [Simple Icons](https://github.com/simple-icons/simple-icons) - ব্র্যান্ড আইকন লাইব্রেরি
- [Swup](https://github.com/swup/swup) - পেজ ট্রানজিশন লাইব্রেরি
- [Howler.js](https://github.com/goldfire/howler.js) - অডিও প্লেব্যাক লাইব্রেরি
- [audioMotion-analyzer](https://github.com/hvianna/audioMotion-analyzer) - অডিও স্পেকট্রাম ভিজ্যুয়ালাইজেশন লাইব্রেরি
- [Carbon Fields](https://github.com/htmlburger/carbon-fields) - WordPress কাস্টম ফিল্ড ফ্রেমওয়ার্ক
- [PHP-Scoper](https://github.com/humbug/php-scoper) - PHP নেমস্পেস আইসোলেশন টুল
