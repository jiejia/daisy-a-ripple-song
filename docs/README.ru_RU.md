# A Ripple Song

A Ripple Song — это классическая тема WordPress. После установки [плагина для подкастов](https://github.com/jiejia/a-ripple-song-podcast) в теме становятся доступны аудиоплеер, подкаст-виджеты и связанные функции.

<img width="" src="../screenshot.png"/>

## Особенности

- 56 тщательно подобранных цветовых схем темы (26 светлых и 26 темных тем)
- Поддержка интернационализации
- Отслеживание данных
- Плавные переходы между страницами на базе Swup
- Несколько виджетов в стиле темы (часть функций требует плагин подкастов)
- Иммерсивный аудиоплеер (требуется плагин подкастов)
