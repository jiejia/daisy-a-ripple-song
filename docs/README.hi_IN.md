# A Ripple Song

A Ripple Song एक क्लासिक WordPress थीम है। इसका साथी [पॉडकास्ट प्लगइन](https://github.com/jiejia/a-ripple-song-podcast) इंस्टॉल करने के बाद आप ऑडियो प्लेयर, पॉडकास्ट विजेट्स और संबंधित फीचर्स सक्रिय कर सकते हैं।

<img width="" src="../screenshot.png"/>

## विशेषताएं

- 56 खूबसूरती से तैयार की गई थीम रंग योजनाएं (26 लाइट और 26 डार्क थीम)
- अंतरराष्ट्रीयकरण समर्थन
- डेटा ट्रैकिंग
- Swup आधारित निर्बाध पेज ट्रांजिशन
- थीम की शैली के अनुरूप कई विजेट्स (कुछ फीचर्स के लिए पॉडकास्ट प्लगइन जरूरी है)
- इमर्सिव ऑडियो प्लेयर (पॉडकास्ट प्लगइन जरूरी है)
